# CET-4 词汇内化练习系统 (CET-4 Vocabulary Internalization System)

这是一个基于 Web 的英语词汇练习系统，旨在帮助学生通过多种交互式练习（Active/Passive 模式）深度掌握 CET-4/6 级词汇。

## 包含内容 (Contents)

### 1. Active Mode (Active 模式)
- **主题**: Wild Animals & Feeding
- **文件**: `cet4_vocab_exercise.html`
- **特点**: 包含完整的 8 个练习板块（音形义、连线、填空、搭配、语法、听力、微型完型、翻译），适合全面训练。

### 2. Passive Mode (Passive 模式)
- **主题**: Entrepreneurs & Willpower
- **文件**: `cet4_vocab_exercise_entrepreneur.html`
- **特点**: 包含精简的 5 个核心板块，词汇难度更高（CET-4/6 进阶），适合快速内化。

## 如何使用 (How to Use)

1. **直接打开**: 双击 `index.html` 或任意 `.html` 文件在浏览器中运行。
2. **学生注册**: 输入姓名、班级、学号开始练习。
3. **提交成绩**: 完成后点击底部 "Submit & Check"，系统会自动评分并显示参考答案。
4. **复制成绩**: 点击 "复制成绩 (Copy Result)" 按钮，将成绩单发送给老师。

## 部署 (Deployment)

推荐使用 GitHub Pages 进行免费托管，以便生成可分享的链接。

1. 将代码推送到 GitHub 仓库。
2. 在仓库 Settings -> Pages 中选择 `main` 分支作为 Source。
3. 获取生成的链接（例如 `https://username.github.io/repo-name/`）发给学生。
